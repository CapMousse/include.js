Include.js is a tiny javascript loader (.4kb gzipped). Speed up your page loading time with this easy tool.

How does it speed up your loading time? Without Include.js you need to download all javascript files when you first hit the site. If you have jQuery, jQuery UI... it means a lot of files to be downloaded before your page is "ready". With Include.js, you just have one file to download before your page is "ready". All other files are loaded after the page.

Usage

Insert Require.js in the head/body end of your document. It just needs an array of files and optionally a callback to run :

require(files, callback);

A file can be represented by a string (path to the file) or another array wich contains the file path, a callback function when the file is loaded and optionally the object name to be loaded.

[
        'javascript/main.js',
        ['javascript/file.js', function(){}],
        ['javascript/jquery.js', function(){}, '$']
    ]
