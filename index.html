Require.js

A tiny javascript loader with cache control

Require.js is a tiny javascript loader (.5kb gzipped) with cache control, so your not going to load a script already loaded.

Just place it in the head of your document with the list of all files needed to accelerate your page loading.

Usage

Insert Require.js in the head/body end of your document. You need to pass an object with all infos needed to do the magic :

require({
    files : array of files,
    success : optional function launched when all files are successfully loaded,
    complete : optional function launched when all files are parsed,
});

A file can be represented by a string (path to the file) or an other array wich contain the file path, a callback function when the file is loaded and optionally the object name to be loaded.

[
    'javascript/main.js',
    ['javascript/file.js', function(){}],
    ['javascript/jquery.js', function(){}, '$']
]
